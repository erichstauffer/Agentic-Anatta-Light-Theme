Theme Factory

Autonomous Shopify theme engineering CLI by Anatta. Describe what you want in plain English — Theme Factory builds it.

Get Started Documentation
terminal
# Install Theme Factory
$ npm install -g @anattadesign/theme-factory

# Run a task — builds, commits, pushes to GitHub + Shopify
$ tf run ./my-first-task.md
✓ Theme updated and pushed to your-store.myshopify.com

# Or use interactive chat mode
$ tf chat
✓ Chat session started

Features

Plain English

Describe what you want in simple English. Theme Factory translates your intent into production-ready Shopify theme code.

Figma Replication

Point Theme Factory at a Figma file and it replicates the complete design as a functional Shopify theme with 96% accuracy.

90% Faster

Reduce Shopify theme development time by 90%. What used to take days now takes minutes with autonomous code generation.

Auto Content

Automatically updates theme content. Push changes to Shopify and GitHub in a single pipeline run.

Shared Memory

Remembers client-specific development patterns and shared learnings. Gets smarter with every project it builds.

Interactive Chat

Use tf chat for interactive task building and live fixes. Build themes conversationally without writing a task file.
